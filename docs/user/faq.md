# Frequently asked questions (players)

## Do I need to install anything?

No. Use a normal web browser and the **player URL** your operator provides.

## Where do I create an account?

On the **player frontend** registration page. See [Getting started](getting-started.md).

## Why can’t I access admin tools?

Administration uses a **separate application**. Only accounts with admin roles see it.

## Is the AI “writing the story” by itself?

AI **assists** turns, but **committed** story state follows **runtime rules** and validation—not unchecked model output. See [How input affects the experience](how-input-affects-the-experience.md).

## The play page won’t connect—what now?

Check your network, try a refresh, then contact your **operator**. They verify the play service and URL configuration.

## Where is forum help?

See [Forum player guide](forum-player-guide.md).
