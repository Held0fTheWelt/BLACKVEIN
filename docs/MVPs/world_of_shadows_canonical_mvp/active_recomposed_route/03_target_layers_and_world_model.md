# Target layers and world model

## Why this document remains necessary

The older MVP family carried much more of the broad WoS target than the active proof-bearing slice currently replays.
Those concept families still matter and must stay visible.

## The five-layer runtime target still carries

### Layer 1 — data, canon, and memory foundations
Structured authored truth, memory storage, provenance, lineage, slotting, thresholds, and conflict-aware retention.

### Layer 2 — semantic runtime
Interpretation, retrieval, relevance, context assembly, temporal validity, entity identity, and hybrid lookup discipline.

### Layer 3 — narrative intelligence
Scene direction, dramatic planning, continuity shaping, transition pressure, and consequence-bearing move handling.

### Layer 4 — emotional and social intelligence
Emotional pressure, social geometry, relationship heat, coalition shifts, room/object resonance, and reaction deltas.

### Layer 5 — consciousness / meta-strategy
Self-preservation, concealment, framing, strategy, hesitation, rebound, suppression, and broader reflective behavior.

## World model families that remain canonical target truth

- world-state and relationship memory
- counterfactual and conflict-bearing memory
- rumor/legend and suppression/recovery memory
- effect surfaces and transformation maps
- recovery intelligence and failure-as-story
- NPC/world simulation
- multi-session continuity
- learned-policy shadow lane

## Honest status reading

These are **not** all equally implemented or freshly proven in the active repository slice.
But they are still:

- source-supported,
- materially relevant,
- and dangerous to lose in later summary passes.

## How to read them now

- treat them as **enduring target truth**,
- tie them where possible to active slice readouts already visible today,
- and keep them separate from false claims of completed proof.

## Active-slice bridges already visible

The active slice already shows part of this broader system through:

- social weather and social geometry readouts,

That partial visibility is why the broader layers still belong inside the MVP instead of outside it.
