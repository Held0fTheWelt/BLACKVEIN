# World of Shadows canonical MVP re-audit package

This archive is a **documentation-first canonical MVP package** produced from a preservation-first re-audit of the supplied World of Shadows MVP archives and their related evidence.

It does **not** pretend to be a full rebuilt repository.
It is the cleaned canonical MVP document set plus the supporting audit material needed to show:

- what was audited,
- what was preserved,
- what had been underrepresented,
- what was integrated now,
- what remains residue,
- and what still blocks a stronger runtime-proof verdict.

## Package contents

- `docs/MVPs/world_of_shadows_canonical_mvp/`
  - the canonical MVP document family
- `docs/audit/world_of_shadows_canonical_mvp_reaudit_2026-04-20/`
  - the master report, source inventory, preservation matrix, improvement review, residue record, runtime-proof task list, changed-files record, and machine-readable companions
- `evidence/raw_test_outputs/`
  - direct pytest rerun outputs captured during this re-audit

## Scope judgment

This package is stronger than the prior six-document canonical set because it restores additional material that was still source-supported but too compressed there:

- enduring target runtime layers,
- writers' room and authoring platform expectations,
- operations / deployment / performance / localization constraints,
- explicit runtime maturity profiles and gate language from v17-v18,
- and a more explicit anti-loss carry-forward map across the supplied archive family.
