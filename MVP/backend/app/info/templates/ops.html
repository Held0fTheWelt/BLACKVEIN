Health endpoints

  • GET /health — simple JSON {"status": "ok"} on the web blueprint (load balancers).
  • GET /api/v1/health — JSON health under the versioned API (rate-limited with other v1 routes).

Docker Compose vs bare metal

Root docker-compose.yml maps the backend container port to 8000 on the compose network while bare-metal Flask often uses 5000. Frontends must set BACKEND_API_URL to reach the correct host/port (e.g. http://backend:8000 inside compose). Details: docs/operations/RUNBOOK.md and README.md.

Environment

See backend/.env.example for database, JWT secrets, FRONTEND_URL, CORS, mail, and play-service variables. Production requires strong SECRET_KEY and JWT_SECRET_KEY (32+ bytes) and email-verification policy as enforced in create_app.

Diagnostics & runbook

  • docs/operations/RUNBOOK.md — startup order, ports, troubleshooting.
  • docs/architecture/backend_runtime_classification.md — what is and is not authoritative runtime.
  • Structured logging uses package loggers under app; trace id header X-WoS-Trace-Id on API responses.
