# Postman sub-suites (per OpenAPI tag)

Each **`WorldOfShadows_Suite_<TagSlug>.postman_collection.json`** file is a **tag-scoped** slice of the API, generated by Postmanify from `docs/api/openapi.yaml`.

Use these for **focused** imports (e.g. only **Forum** or only **Auth**) during development or reviews.

Regenerate all files with:

```bash
python -m postmanify.tools generate
```

See also `postman/postmanify-manifest.json` for the last generation metadata.
