# Built MVP Status

This package contains a working autark fy-suite MVP with:
- internal evidence registry
- internal semantic index
- internal run journal
- model router policy
- shared context pack service
- complete suite adapters for contractify, testify, documentify, docify, despaghettify, dockerify, and postmanify
- init / inspect / audit / explain / prepare-context-pack / compare-runs / clean / reset support

All internal state remains inside the fy workspace. The adapters work outward against a target repository.
