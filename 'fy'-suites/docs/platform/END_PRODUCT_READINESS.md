# fy End Product Readiness

This workspace now includes the product-facing closing surfaces for the defined autark fy suite scope.

## What to regenerate before release

- `fy-platform suite-catalog`
- `fy-platform command-reference`
- `fy-platform export-schemas`
- `fy-platform doctor`
- `fy-platform final-release-bundle`

## What these outputs prove

- the current suite inventory is explicit,
- the stable lifecycle and native commands are documented,
- shared contracts are exported as JSON-schema-style files,
- workspace health is summarized in one doctor view,
- and the final release bundle ties release readiness and production readiness together.
